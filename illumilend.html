IllumiLend - Augmented Storage Room

A color-guided system helping students quickly find and return reserved equipment.

Overview

IllumiLend transforms chaotic university equipment rooms into intuitive, self-service spaces. Students swipe their ID, follow colored floor arrows, and pick up items from illuminated cubbies—all synchronized with mobile and wall displays. Testing with 10 users showed 80% found the color paths faster, and 60% were ready to use it immediately.

The Problem

Digital reservation systems efficiently handle bookings but abandon users at the storage room door. Users waste time searching shelves and leave uncertain whether the system registered their actions.

01
No Physical Wayfinding

Digital systems show what you reserved, not where it is. Users hunt through shelves translating codes like "B-07" to physical locations.

02
Missing Confirmation Loop

No system acknowledgment after pickup or return. Users leave uncertain whether actions registered.

03
No Automated Verification

Misplacements surface late. Staff manually reconcile inventory; disputes arise without evidence trails.

Design Challenge

How might we make the storage environment itself guide and confirm user actions? The gap isn't in reservation technology—it's in the physical experience once users enter the room.

User Research

Observations & Analysis

At Télécom Paris equipment room, we observed: students book online but receive minimal location details, manually scan labels to match codes, take items without system confirmation, and staff manually check returns.

Competitive Analysis

Library Systems
RFID checkout but no wayfinding to shelves
Warehouse Picking
Handheld scanners with routes—not self-service
Smart Lockers
Automate retrieval but need heavy infrastructure
AR Navigation
Requires phone-holding—high cognitive load

Key insight: No solution combined ambient guidance (hands-free), personalized color mapping, and immediate verification for university equipment rooms.

Core Design Principles

Make the Room Smart, Not Just the App

Embed guidance in the physical environment—floor projections and shelf lighting create an ambient interface.

Color as Primary Navigation Language

Color is processed pre-attentively. Users see "their blue" instantly across room, shelf, and screen.

Multi-Modal Redundancy

Wall screen mirrors mobile content for users without phones, dead batteries, or who prefer larger displays.

Design Process

Stage 1: Hand-drawn room layouts with colored markers simulating light patterns. Validated with 3 team walkthroughs.

Stage 2: Digital wireframes in Figma showing initial UI layouts.

Final System Components

Badge Entry: Door scanner reads ID, pulls reservation, assigns color (<2 seconds)

Floor Navigation: Overhead projector casts directional arrow in user's color

Shelf Illumination: LED strips embedded in shelf edges light specific cubbies

Stage 3: Interactive UI mockups in Figma—reservation instructions, item details with colors/states/due times, new reservation flow.

Conclusion

Outcome

IllumiLend proves the room itself can communicate. By embedding guidance in the physical environment, we eliminated location friction—users found items 80% faster (subjective), and no one picked wrong items when lights were active. The system offers reduced staff burden, better user experience, scalable foundation, and built-in accessibility.

Key Refinements

✓ Keep: Color-Guided Navigation

80% relied heavily on it. Refinements: Avoid red for placement (reserve for errors), cap simultaneous users (4-6 colors max), always pair color with text labels/IDs for accessibility.

✓ Add: Explicit Confirmation Feedback

70% hesitation shows users need closure. Solution: After pickup/return: cubby light changes state (blue→green), soft audio ping (optional), progress indicator updates ("1/2 picked up"). No modals or extra taps—keep it ambient.

✓ Future: IoT + Computer Vision

Next phase: RFID tags on items + cubby sensors detect events. Overhead camera for exception review. Automated condition logging. Lights guide; sensors verify.
